### 8. Memory Correction (Not Deletion)

**Philosophy:** Memories are corrected, never deleted.

**Correction Flow:**
1. View original memory
2. Edit text/category/entities
3. New version created
4. Original marked as "corrected"
5. Full audit trail preserved

**Why This Matters:**
- History is preserved
- Can see how understanding improved
- No accidental data loss
